Projects

Here are some of the projects I've developed:

Applications/tools

  • VAST XML Parser - an XML parser that extracts Creative and MediaFile element metadata from Video Ad Serving Template XML tags.

  • CSV to Snowflake CLI - A command-line tool for uploading CSV data to the Snowflake data warehouse.

  • file_retirement CLI - A simple command-line script for cleaning up old, duplicate files. Intended use is for automation via Crontab, Windows Task Scheduler, etc.

Automation/bots

  • Gutenberg Sci-Fi Twitter Bot - a Twitter bot that posts daily recommendations of science fiction books from Project Gutenberg's collection of public domain works. Fully automated via Github Actions.

  • The Simpsons - Episode History Twitter Bot - A Twitter bot that posts every day (when possible) about an episode of The Simpsons that aired that day, however many years ago. Fully automated via Github Actions.
