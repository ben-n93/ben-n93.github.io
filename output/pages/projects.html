projects

Here are some of the projects I've developed:

APIs

  • Archer API is a REST API based on the adult animated sitcom Archer. It provides data about Archer's characters, episodes and quotes.

    • Built with Flask and hosted on Heroku. You can view the source code on Github.

Applications/tools

  • geojson-shave - A command-line tool that reduces the size of GeoJSON file by truncuating coordinate values to the specified decimal points and eliminating whitespace.

  • tab-pal - A command-line application that makes it easier, quicker and prettier to create custom colour palettes in Tableau.

Automation/bots

  • Gutenberg Sci-Fi Twitter Bot - a Twitter bot that posts daily recommendations of science fiction books from Project Gutenberg's collection of public domain works.

    • You can read about how I created this project on my blog.

Data analysis

  • I have analysed and visualised different data sets, primarily using Juypter Notebooks and the pandas library. Some of my favourite projects include:

    • Which Sydney LGAs have the lowest and highest unemployment rates
    • Using Spotify's API to analyse my Favourite Rock playlist
    • Where are Australia's oil and gas pipelines located?
